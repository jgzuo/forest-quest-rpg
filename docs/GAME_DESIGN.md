# Forest Quest RPG - 游戏设计文档

## 🎮 游戏概述

**游戏名称：** Forest Quest（森林探险）
**游戏类型：** 2D像素风格动作RPG
**技术栈：** Phaser.js 3.80+
**目标平台：** Web浏览器（桌面端优先）

---

## 📖 故事大纲

### 背景故事
在一个宁静的森林王国里，黑暗力量突然苏醒。原本守护森林的树妖们被腐化，地底的鼹鼠军团也变得疯狂攻击村民。年轻的英雄**艾瑞克**（Eric）站了出来，誓言要拯救森林。

### 主要角色
- **艾瑞克（Eric）**：主角，勇敢的年轻战士
- **村长**：发布任务的老智者
- **NPC们**：提供线索和帮助的村民

### 游戏章节
1. **第一章：黑暗降临**
   - 从家乡小镇出发
   - 学习基本操作
   - 击败腐化的鼹鼠

2. **第二章：森林深处**
   - 进入迷雾森林
   - 对抗树妖战士
   - 收集净化宝石

3. **第三章：洞穴探险**
   - 探索地下洞穴
   - 发现黑暗力量的源头
   - 最终决战：树妖王

## 🎯 游戏机制

### 核心玩法
- **移动**：WASD 或 方向键
- **攻击**：空格键（近战攻击）
- **交互**：E键（与NPC对话、开宝箱）
- **暂停**：P键

### RPG系统
1. **战斗系统**
   - 近战攻击（剑）
   - 伤害计算：基础伤害 + 武器加成
   - 暴击系统（2倍伤害，15%概率）

2. **成长系统**
   - 经验值（XP）
   - 等级提升（增加生命值、攻击力）
   - 技能树（后期扩展）

3. **道具系统**
   - 💎 宝石：净化森林的关键道具
   - 🪙 金币：购买装备和道具
   - ❤️ 生命药水：恢复生命值
   - ⚔️ 武器升级：提升攻击力

4. **任务系统**
   - 主线任务：推进剧情
   - 支线任务：获得额外奖励
   - 成就系统：解锁特殊奖励

## 🗺️ 游戏世界

### 区域设计

#### 1. 小镇（安全区）
- **功能**：起点、存档点、商店
- **NPC**：村长、商人、其他村民
- **建筑**：主角家、商店、村长屋

#### 2. 森林区域（战斗区）
- **环境**：树木、岩石、灌木、瀑布
- **敌人**：鼹鼠、树妖战士
- **道具**：宝石、金币、药水
- **特殊地点**：瀑布后的秘密洞穴

#### 3. 洞穴区域（Boss战）
- **环境**：黑暗、岩浆、水晶
- **敌人**：精英鼹鼠、树妖守卫
- **Boss**：树妖王（多阶段战斗）

---

## 👹 敌人设计

### 1. 鼹鼠（Mole）
- **生命值**：30
- **攻击力**：5
- **移动速度**：慢
- **特点**：成群出现，从地下突然钻出
- **弱点**：攻击时暴露

### 2. 树妖战士（Treant Warrior）
- **生命值**：80
- **攻击力**：12
- **移动速度**：中
- **特点**：会召唤小树苗
- **弱点**：火焰攻击（后期扩展）

### 3. 树妖王（Treant King）- Boss
- **生命值**：500
- **攻击力**：25
- **移动速度**：慢但致命
- **特殊技能**：
  - 根须缠绕（束缚玩家）
  - 落石攻击（范围伤害）
  - 召唤树苗（召唤小怪）
- **战斗阶段**：
  - 第一阶段：普通攻击
  - 第二阶段（50%HP）：使用特殊技能
  - 第三阶段（20%HP）：狂暴模式
